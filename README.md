# Econland — Simulador de Treino

Reconstrução **não-oficial** do Macroeconomics Simulation: Econland, para treino
antes das tentativas oficiais do Executive MBA.

- 7 anos, 4 decisões (taxa de juro, IRS, IRC, despesa pública), aprovação 0–100.
- Motor macro próprio, calibrado contra uma corrida real do cenário Base Case.
- O conteúdo está **encriptado**; o acesso requer palavra-passe (partilhada na turma).

Abre o link do GitHub Pages e introduz a palavra-passe.

> Ferramenta educativa entre colegas. Não substitui o simulador oficial nem
> reproduz o seu código proprietário.
